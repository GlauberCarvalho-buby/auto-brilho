Auto Brilho
Aplicação web construída com TypeScript e Vite, focada em desenvolvimento React avançado (Senior React Dev). Este repositório contém o código-fonte da aplicação, configurações de build e arquivos de suporte.

Requisitos
Antes de começar, instale:

Node.js (versão LTS recomendada)
npm (já vem com o Node.js) ou yarn/pnpm, se preferir
Inatalação
Baixe o projeto: https://github.com/GlauberCarvalho-buby/auto-brilho.git cd SeniorReactDev

Instale as coisas: npm install

Rode o projeto: npm run dev

Vai abrir em http://localhost:5173

O que tem aqui
Vite pra build rápido
TypeScript
Tailwind CSS
Rotas definidas em routes.ts
Configurações em vite.config.ts e tailwind.config.ts
Pra fazer build
npm run build fica na pasta dist/.
